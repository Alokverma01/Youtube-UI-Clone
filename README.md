# YouTube UI Clone

A minimal YouTube-style UI built with React, TypeScript and Vite. This repo demonstrates a lightweight, component-driven layout including a top navigation bar and a left sidebar — a starting point for building media-focused interfaces.

## Features

- Responsive top navigation (`src/components/navbar.tsx`)
- Left sidebar with navigation items (`src/components/sidebar.tsx`)
- Simple layout using Vite + React + TypeScript

## Tech Stack

- Vite
- React
- TypeScript
- Plain CSS (see `src/index.css`)

## Prerequisites

- Node.js 18+ or compatible
- npm (or pnpm/yarn)

## Setup

1. Install dependencies

```bash
npm install
```

2. Run the development server

```bash
npm run dev
```

3. Build for production

```bash
npm run build
```

4. Preview the production build

```bash
npm run preview
```

## Project Structure

- `index.html` — App entry
- `src/main.tsx` — App bootstrap
- `src/App.tsx` — Main layout
- `src/components/navbar.tsx` — Top navigation
- `src/components/sidebar.tsx` — Left sidebar
- `src/index.css` — Styles

## How to Contribute

- Open an issue describing the change or feature.
- Send a PR with focused, small commits.

## Notes

- This is a UI-focused clone for learning and experimentation. It does not provide video playback or backend APIs.
